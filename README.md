# Miniaturized-ADAS
Advanced driver-assistance systems (ADAS) educational platform based on Infineon's sensing technologies like RADAR or 3D-ToF-LIDAR camera.

The platform consits of 3 parts:
- Lego Radar Train
- Radar Model Car
- Audi Q2
